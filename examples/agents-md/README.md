# AGENTS.md Examples

Copy-paste templates for a project's root `AGENTS.md` when this repository's skills are installed. Pick the folder matching the project's setup and adapt the routing rules to the project.

| Folder | Scenario |
| --- | --- |
| `standalone/` | A project using this repository's skills only |
| `with-agent-team/` | Same, plus the `agent-team` CLI (github.com/tae2089/agent-team) as the durable work ledger |

These are examples for *other* projects — this repository's own guidance lives in the root `AGENTS.md`.
